# [*gupi*](## "Go User Pacman Interface")

#### A modern, interactive TUI for managing Arch Linux & AUR packages, built in Go.

---

Stay tuned! This is a rewrite and improvement upon [PacLean](https://github.com/eleinah/paclean)

Building this with [Charm tools](https://charm.sh/libs) and Go.

---

<ins> Planned functionality </ins>

<p>ğŸ”¹ Browse and search official Arch Linux repositories and the AUR in a unified TUI</p>
<p>ğŸ”¹ Install packages from both official repos and the AUR</p>
<p>ğŸ”¹ Remove and reinstall packages</p>
<p>ğŸ”¹ Clear pacman cache</p>
<p>ğŸ”¹ View detailed package information</p>
<p>ğŸ”¹ Visualize dependencies and orphans</p>
<p>ğŸ”¹ Execute batch operations (install, remove, clean)</p>
<p>ğŸ”¹ User-friendly keybindings and theming</p>
